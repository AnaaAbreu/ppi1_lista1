Este site foi desenvolvido como parte da disciplina de Programação para Internet e reúne os principais conteúdos estudados sobre a Internet, a World Wide Web, o protocolo HTTP e outros conceitos fundamentais, além de páginas de currículo e contato para praticar o desenvolvimento de sites utilizando HTML.



Diferença Internet/Web

A Internet é uma rede mundial que conecta computadores e dispositivos. Já a Web é um serviço que utiliza essa rede para disponibilizar páginas acessadas por navegadores por meio do protocolo HTTP.
