História da Internet e da Web



Linha do tempo

  1. Surgimento das redes de computadores

    Na década de 1950 surgiram os primeiros computadores de grande porte. Com o passar do tempo surgiu a necessidade de conectar essas máquinas para compartilhar informações e recursos.

  2. ARPANET

    A ARPANET foi criada no final da década de 1960 e é considerada a precursora da Internet moderna. Seu objetivo era manter a comunicação funcionando mesmo que parte da rede fosse interrompida.

  3. TCP/IP

    Em 1973 iniciou-se o desenvolvimento do protocolo TCP por Bob Kahn e Vinton Cerf. Em 1978 surgiu a divisão entre TCP e IP, formando a suíte TCP/IP utilizada até hoje.

  4. DNS

    O DNS foi criado em 1984 para transformar nomes de sites em endereços IP, facilitando a navegação na Internet.

  5. Criação da World Wide Web

    Em 1990, Tim Berners-Lee, pesquisador do CERN, criou a World Wide Web (WWW), o protocolo HTTP e a linguagem HTML.

  6. HTML

    O HTML (HyperText Markup Language) é a linguagem utilizada para estruturar páginas da Web.

  7. HTTP

    O HTTP (HyperText Transfer Protocol) é o protocolo responsável pela comunicação entre navegadores e servidores Web.

  8. Navegador

    O navegador é o programa utilizado para acessar páginas da Web, interpretando arquivos HTML enviados pelo servidor.

  9. Servidor Web

    O servidor Web armazena os arquivos de um site e responde às requisições feitas pelos navegadores através do protocolo HTTP.

  10. Web 1.0 x Web Moderna

    A Web 1.0 era composta principalmente por páginas estáticas, enquanto as aplicações Web modernas permitem interação com o usuário, processamento de dados e atualização dinâmica das informações.
